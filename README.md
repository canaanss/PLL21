# PLL 连拧训练

一个纯静态 CFOP PLL 训练网站，双击 `index.html` 即可打开。

## 功能

- 21 个 PLL 公式窗口
- 每个 PLL 有识别示意图
- 一键生成 21 个 PLL 随机顺序
- 整组连拧计时
- 点击任意 PLL 进入单公式干净计时界面
- 上一个 / 下一个训练控制
- 本地保存上次用时和最佳用时

## 快捷键

- 主界面空格：开始 / 停止连拧计时
- 单公式界面空格：开始 / 停止单公式计时
- 单公式界面 Esc：返回主界面
- 右箭头：下一个 PLL
- 左箭头：上一个 PLL

## 说明

当前示意图为网页内置 SVG 识别图，不依赖外网图片。公式可以在 `app.js` 的 `PLLS` 数组里按你的手法习惯替换。
